Introduction to CSS

CSS?

CSS (Cascading Style Sheets) is a styling language used to style a web page. It describes how HTML elements are to be displayed on screen, paper, or in other media.

CSS saves a lot of work. It can control the layout of multiple web pages all at once.

Styling

When a browser reads a style sheet, it will format the HTML document according to the information in the style sheet.

There are three ways of inserting a style sheet:

  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS

Inline styling may be used to apply a unique style for a single element.

To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property. 


  This is a <span style="color: red; font-style: italic;">Container</span>

Internal CSS

An internal style sheet may be used if one single HTML page has a unique style. The internal style is defined inside the <style> element, inside the head section. 


  <style>
    span {
      color: red;
      font-style: italic;
    }
  </style>

External CSS

With an external style sheet, you can change the look of an entire website/page by changing just one file. Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the head section. 


  <link rel="stylesheet" href="styles/index.css">

CSS Syntax

Selectors

Selectors are patterns used to match, or select, the elements you want to style.

Universal Selectors

Universal selector * matches of any type.


  * {
    color: red;
  }

Type Selectors

Type selectors match elements by node name. In other words, it selects all elements of the given type within a document. 


  p {
    color: red;
  }

ID Selectors

ID Selectors match an element based on the value of the element's ID attribute. In order for the element to be selected, its id attribute must match exactly the value given in the selector. 


  #base {
    color: lime;
  }

Class Selectors

Class selectors match elements based on the contents of their class attribute. 


  .italic {
    font-style: italic;
  }

Attribute Selectors

Attribute selectors match elements based on the element having a given attribute explicitly set, with options for defining an attribute value or substring value match. 


  p[name="custom"] {
    color: yellow;
  }

Pseudo Selectors

A pseudo is a keyword added to a selector that lets you style a specific state of the selected classes or elements. 


  h1:hover {
    color: red;
  }

Comments

Comments are used to add explanatory notes to the code or to prevent the browser from interpreting specific parts of the stylesheet. By design, comments have no effect on the layout of a document. 


  /* This is a
    multi-line comment */

  h1 {
    color: red; /* Set text color to red */
  }

Errors

Errors in CSS can lead to unexpected behavior or styles not being applied correctly.

  • Missing semicolons
  • Invalid property names
  • Invalid values
  • Unclosed braces
  • Extra colons or braces

Missing Semicolons


  /* NOT OK */
  p {
    color: red
    text-align: center;
  }

  /* OK */
  p {
    color: red;
    text-align: center;
  }

Invalid Property Name


  /* NOT OK */
  p {
    colr: red;
  }

  /* OK */
  p {
    color: red;
  }

Invalid Values


  /* NOT OK */
  p {
    font-size: -2em;
  }

  /* OK */
  p {
    font-size: 2em;
  }

Unclosed/Extra Symbols


  /* NOT OK */
  p {
    color:: red;

  h1 {
    color: blue;
  }


  /* OK */
  p {
    color: red;
  }

  h1 {
    color: blue;
  }

Tips

  • Use a code editor with syntax highlighting
  • Validate your CSS with a CSS linter or validator.
  • Write CSS in small sections and test frequently.

RGB

The rgb() function is used to generate a color from red, green, and blue values. The function has three parameters, each allocating per primary colors (red, green, and blue). Values of each parameters range from 0 to 255. 


  background-color: rgb(255, 0, 0); /* red */
  background-color: rgb(0, 255, 0); /* green */
  background-color: rgb(0, 0, 255); /* blue */

Hex

To specify a color using hexadecimal, prefix a hashtag (#) prior to the value. Numbers 0-9 and letters A-F are used, each corresponding to their numerical value. 


  background-color: #f00; /* red */
  background-color: #0f0; /* green */
  background-color: #00f; /* blue */

HSL (Hue, Saturation, Lightness)

The hsl() function is used to generate a color using HSL values. Hue is the degree on the color wheel (from 0 to 360); Saturation is a percentage value: 0% means a shade of gray, 100% is the full color; Lightness is a percentage: 0% is black, 50% is neither light or dark, 100% is white. 


  background-color: hsl(0, 100%, 50%); /* red */
  background-color: hsl(120, 100%, 50%); /* blue */
  background-color: hsl(240, 100%, 50%); /* green */

Background Color

To set a background color, the background-color property is declared.


  background-color: blue;

Text Color

To set a color to a text, color property is declared.


  color: red;

Border Color

To set a color to the border, border-color property is declared.


  border: 2px solid;
  border-color: limegreen;

Backgrounds

The background property sets all background style properties at once, such as color, image, origin and size, or repeat method. This property is a shorthand for the following CSS properties:

  • background-attachment
  • background-clip
  • background-color
  • background-image
  • background-origin
  • background-position
  • background-repeat
  • background-size

Image

background-image property is declared to assign a background image to the element.


  background-image: url('/assets/cat.jpg');

Repeat

background-repeat property sets how background images are repeated.


  background-image: url('/assets/cat.jpg');
  background-repeat: repeat;
  
  background-image: url('/assets/cat.jpg');
  background-repeat: no-repeat;

Sizes

The background-size property sets the size of the element's background image.


  background-image: url('/assets/cat.jpg');
  background-repeat: no-repeat;
  background-size: 50%;

Borders

Boreders are used to define the visual boundaries around HTML elements.

Styles

border-style property is declared to set the border's style.

Width

border-width property is declared to set the border's width.


  border-width: 0.5em; /* all sides */
  border-width: 0.5em 0 0.25em 0.75em; /* top right bottom left */

Color

border-color property is declared to set the border's color.


  border-color: limegreen; /* all sides */
  border-color: limegreen red blue yellow; /* top right bottom left */

Sides


  border-left-width: 0.5em;
  border-left-color: lightblue;

Radius

border-radius property is declared to apply rounded corners to the border.


  border: 4px solid white;
  border-radius: 13px;

  border: 4px solid white;
  border-radius: 50%;

Margins

Margins are properties used to create space around an element, outside of any defined borders. They define the distance between an element's border and the surrounding elements.

margin property is declared to set the element's margin. By default, due to historical reasons, the <body> tag has a default margin of 8px. 


  margin: 25px; /* all sides */
  margin: 25px 50px; /* top-bottom left-right */
  margin: 25px 33px 50px; /* top left-right bottom */
  margin: 25px 33px 50px 67px; /* top right bottom left */

Padding

Padding properties are used to generate space around an element's content, inside of any defined borders.


  padding: 4em;

Height and Width

height and width properties are declared to set an element's height and width dimensions.


  height: 50px;
  width:  100px;

max-width

max-width property sets the maximum allowed width of an element. This prevents the width of an element to be larger than the max-width property value.


  max-width: 50px;
  width: 100px;
